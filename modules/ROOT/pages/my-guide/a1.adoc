= A1-东汉三国

== 小知识

* 我们使用 xref:my-guide/zhenxing.adoc[“ 105 ” 阵型站位]，当然刚开局角色很少，
我们也没什么选择余地，基本都是有什么用什么，这里给出的只是一个目标，后文如果不特殊说明我们就一直使用这个阵型。
* A1过程中我们可能需要频繁切换到主位面，前期跟着主线引导任务进行即可。
* A1的蜀国势力可以提供辅助（方士）技能，东吴势力可以提供弓手技能。
* 前期不建议使用武夫和学者这两个职业，所以后文输出指的就是弓手，坦克指的就是护卫，辅助指的就是方士
* 不同职业可以学习的技能不同类型可以点击 xref:my-guide/zhiye-jineng.adoc[这里] 查看
* 学习技能时如果技能上的标签不匹配我们的职业，那我们就不可以学习。
* 我们可以通过转职书进行转职，转职要求在转职界面可以查看。
* 转职并非不可逆，达到要求即可重新转职其它分支，之前分支的技能虽然不能继续使用，但也不会丢失，转职回来仍然可以使用。
