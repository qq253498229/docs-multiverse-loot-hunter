= 为什么我推荐赵云做辅助

* 赵云的基础属性是：（ *勇71* + 智33 + 体47 + *敏73* + 精27），很多玩家都选择让赵云做输出
* 但我为什么推荐赵云做辅助呢？原因也正是因为他的敏太高了
* 很多玩家在游玩的过程中都会使用一个套路，就是辅助先出手给输出加能量，然后输出放技能打伤害
* 但赵云的敏太高，很容易出现比辅助还快的情况，有时候甚至需要赵云拖鞋、下灵兽等手段降低速度
* 据我测试大概在A7-二战风云位面，辅助学习了【山姆梦+灯塔辉光+星条荣耀】三个技能后，速度才能真正超过赵云
* 而在A3开始，辅助就要开始给输出加能量了。这中间的过程说长不长，说短也不短
* 而主角渡过前期的乏力阶段，逐渐开始显示战斗力，再加上沙漠之鹰的超高伤害，战云的输出又显得不那么重要
* 既然赵云迟早是要转T或者辅助的，那还不如从头开始就直接做辅助，省着后面还要转职浪费职业点数
* 而且在这中间经常要对比赵云和辅助的速度，换一次装备就要关注一次，输出不能太高辅助不能太低，头发都要掉了
* 而如果主角做输出，其他角色做辅助的话，只需要辅助尽可能的堆速度就行了。
* 因为主角随时可以洗点，完全可以让辅助做到极限速度，然后主角临时将敏调整到比辅助低上几点，实现全队速度最大化

当然这些都是我的个人建议，赵云做输出、辅助、坦克都是可以的，具体选择由大家自行决定。
