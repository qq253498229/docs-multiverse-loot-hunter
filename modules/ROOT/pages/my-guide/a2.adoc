= A2-武侠江湖

== 小知识

* 每通关一个位面的一个难度，都会获得一次星魂奖励
** xref:my-guide/xinghun-duihuan.adoc[星魂能兑换的东西] 很多，可以自行选择
* A2的少林阵营提供坦克技能，武当阵营提供辅助技能
* 坦克的护盾技能受法攻影响，所以从现在开始，坦克就需要改堆法攻了
* 令狐少侠和吕布类似，都是前期即可获取的能T能输出的角色。具体是哪个T哪个输出，还是都做T自行决定

== 待办事项

* 在进入这个位面之前，最好将之前位面的下一个难度都通关一遍，同时 xref:my-guide/a1-zhuangbei-tihuan.adoc[更新装备]
* 将本位面声望刷满
* 招募本位面全部角色
* 更新副官以及主位面工作NPC
* 通关本位面
