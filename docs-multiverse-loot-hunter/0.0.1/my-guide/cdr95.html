95%CDR-双奶单输出玩法

这种玩法的辅助一定要比输出快，所以辅助要多练一些被动加速度的技能

  • A7第10关卡掉落的胜利套装效果是（技能冷却+15%）

  • A7伦敦商店购买的文明套装图纸（技能冷却+18%）

  • A2武当山购买的真武套装图纸（技能冷却+15%）

  • 这样三个套装叠加技能冷却就已经达到48%了，再在其他部位找一些技能冷却词条即可接近95%CDR

因为堆满95%CDR前期比较困难，所以采用次一档的83%，这样使用铁箭雨+天下布武两个技能正好达成循环。

  • 但是由于使用了铁箭雨需要5点能量，那么光靠回合1+副手1+两个辅助2，这么算才4点能量，那该如何解决呢？

  • 要么增加辅助的回能，要么增加辅助的数量（还要多做一套装备），这里选择增加辅助的回能

    • 辅助也要带回能副手，这样每两回合就多出2点能量够放一个武当九阳功，也就完成了整套流程

    • 所以这个配置至少需要2个A1大副本出的回能副手，输出一个，辅助一个（另外的辅助不带）

最终目标

  • 火炮需要83%技能冷却，除武器和副手之外，都要洗出技能冷却词条，尽量堆高输出

  • 辅助需要72%技能冷却，除武器和副手之外，都要洗出技能冷却词条，速度比输出快

  • 战斗流程为：辅助充能→天下布武→辅助充能→铁箭雨→循环

具体实现方法

注意下面步骤可以SL，请务必提前备份存档，防止操作失误。如果不想SL可以多积累一些锻造点来洗词条

  • 首先做好准备工作

    • 去A7第10关多刷点首饰、布甲、布鞋，最好刷满背包

    • 去A7伦敦城买蓝色衣服+蓝色鞋+紫色衣服+紫色鞋，一共4张图纸

    • 去主位面锻造台锻造装备，分别做20件蓝色盟军套装衣服以及鞋子，然后保存游戏

  • 开始做衣服

    • 将这20件蓝色衣服升级到紫色文明套装，然后筛选技能冷却词条，需要数值大于5.5%，如果不出就读档重来

  • 然后是鞋子

    • 和衣服不同的是，鞋还需要核心词条中的速度大于105%才能用，同样如果不出就读档重来

  • 最后是真武套装

    • 去A2第9关刷头盔和护腕底材

    • 真武套装是蓝色的，所以在黄装阶段就要保存

    • 同上，不出技能冷却词条就读档重来

  • 命盘中紫色孔插满冷却宝石

    • 命盘孔用大副本掉落的命魂解锁

    • 命石在秘境产出，在主位面合成升级

  • 灵兽技能也有技能冷却加成，不过前期功勋不够，不强求
