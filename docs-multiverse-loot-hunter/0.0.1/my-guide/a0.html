开局提示

  • 进游戏先 把手枪上锁！把手枪上锁！把手枪上锁！ 这玩意贼猛，丢失会影响前中期进度

  • 设置输入代码6666领取赵云，非常强力，能用到毕业

  • 前期武夫和学者很弱，选择角色定位的时候尽量选择护卫（坦克）、弓手（输出）和方士（辅助）

  • 虽然这是一个挂机游戏，但本质上也是一个回合制游戏。

    • 回合制游戏最重要的就是敏序，出手顺序有时候就决定战斗是胜利还是失败

    • 所以渡过开局角色较少的阶段后，我们培养角色的首要条件就是敏捷不能太低

    • 一般看到40以下敏捷的，基本就不会考虑进入主力阵容了。只会用来派遣，或者安排在城镇进行工作
