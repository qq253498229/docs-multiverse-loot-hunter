105阵型

  • 游戏开始阶段我们会使用"105"阵型站位

  • "105"意思是第一排站1个角色，第二排不站任何角色，第三排站5个角色。

  • 因为游戏战斗中的目标机制是：如果面前有敌人（针对横排）则会首先攻击面前的敌人； 如果没有则会攻击最上面一排。

  • 所以上路单位会承担大部分伤害，有经验的玩家会在培养角色的时候， 专门培养“坦克”（指相对于输出更偏向于抗打击能力）型角色放在第一排和第三排的最右边。
