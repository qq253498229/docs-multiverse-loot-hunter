A7-二战风云

小知识

  • 这个位面解锁了游侠职业，游侠有两个超强单体技能【绝命狙击】、【弹药箱】

    • 20层弹药+狙击，能秒杀比自己高30级的BOSS

  • 这个位面解锁了医生职业，方士辅助可以转职

  • 第一关卡（BOSS曼施坦因）掉落的步枪自带速度词条，附加词条也可以出速度，是当前速度最快的武器

    • 输出和辅助都应该带这件武器，辅助因为功能已经变为回能量，所以法攻也就不是那么重要了

    • 带沙漠之鹰的输出不用换，还是要带沙漠之鹰。沙鹰可以在锻造洗出3条速度

  • 这个位面有两套技能冷却套装，再加上A2武当山也有一套，可以体验一下 95%CDR-双奶单输出玩法

  • 如果之前没有刷够灵兽，这个位面学习了【绝命狙击】、【弹药箱】后，伤害完全够秒杀50级1阶灵兽的（5000万血）
