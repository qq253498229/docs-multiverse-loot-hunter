A1-大副本-火烧赤壁

小知识

  • A1的物价最便宜，所以尽量在A1买门票、秘境券、刷新券等

    • 商店买门票→刷副本→装备卖钱→商店买门票，这样循环

    • 在后期科技加成下，光靠大副本的收益就够门票钱，实现无限循环

    • 之后我们可能要经常在A1挂机，因为这里获取门票的效率最高

  • 拥有+1能量的副手后，主力输出更换为铁箭雨速刷大副本

    • 大副本每一个阶段分别算作一场新的战斗，技能冷却会被重置，所以每个阶段开始都可以施放一个铁箭雨

    • 输出足够给力的情况下甚至会一路秒杀过去

    • 如果有两个回能副手，那么可以让速度慢的输出改用辕门射，这个技能单体伤害更高，用来在高速输出清光小怪后点杀boss

  • 前期大副本的小怪有一定难度，尽量提高输出的速度，这样才能在小怪出手前放技能，减少坦克的压力

  • 大副本获得的钥匙攒起来，等集齐100个，之后可以SL开启。

    • 如果开除的淬炼石少于5个则读档重新开启，直到满意为止

    • 也可以针对图纸进行SL

  • 现在开始需要关注物品掉落词条了

    • 大副本150%物品掉落，必出宝箱钥匙

    • 秘境400%物品掉落，必出命石

    • 派遣900%物品掉落，必出材料

    • 大地图1900%物品掉落，必出秘境门票

    • 掉落超出部分无效

待办事项

  • A1大副本的目标就是获得能量回复+1的副手，最少要1个给主力输出用。越多越好，最好主队一人一个
